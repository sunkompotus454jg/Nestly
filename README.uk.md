# Nestly

[English](README.md) | [Українська](README.uk.md)

Nestly — це преміальний, легкий та сучасний інструмент для організації робочого столу у Windows. Він дозволяє групувати ваші ярлики, файли та папки у настроювані сітки (fences) безпосередньо на робочому столі, допомагаючи підтримувати чистоту та порядок у робочому просторі.

## Особливості

*   **Сітки на робочому столі:** Створюйте вікна, розмір і положення яких можна змінювати, для зберігання елементів робочого столу.
*   **Преміальний інтерфейс:** Безрамковий дизайн, власні тіні та сучасні елементи стилю glassmorphism.
*   **Автозгортання:** Сітки можуть автоматично згортатися до заголовка, коли курсор миші залишає їх, заощаджуючи простір на екрані.
*   **Керування профілями:** Створюйте та перемикайтеся між кількома профілями (наприклад, Дім, Робота, Ігри) з абсолютно різними макетами та сітками.
*   **Глобальні гарячі клавіші:** Використовуйте комбінації клавіш для швидкого приховування або показу всіх сіток.
*   **Функція пошуку:** Вбудований анімований рядок пошуку для миттєвого знаходження файлів усередині сітки.
*   **Користувацькі теми:** Повністю настроювані кольори, рамки та прозорість фону. Включає вбудований редактор тем.
*   **Інтеграція в системний трей:** Керуйте профілями, отримуйте доступ до налаштувань і створюйте нові сітки прямо із системного трею Windows.
*   **Інтеграція в контекстне меню:** Додає опцію "Створити сітку Nestly" безпосередньо в контекстне меню робочого столу Windows.
*   **Багатомовна підтримка:** Доступно англійською, російською та українською мовами.
*   **Автозапуск:** Опція автоматичного запуску разом із Windows.

## Встановлення

### Використання інсталятора (Рекомендовано)
1. Завантажте останній `Nestly_Installer.exe` зі сторінки релізів.
2. Запустіть інсталятор і дотримуйтесь інструкцій.
3. Інсталятор автоматично налаштує ярлики на робочому столі та інтеграцію в контекстне меню.

### Портативна версія
1. Завантажте портативну збірку `Nestly.exe`.
2. Розмістіть її в будь-якому місці на вашому комп'ютері та запустіть.

## Збірка з вихідного коду

Nestly створено за допомогою Python 3 та PyQt6.

### Вимоги
* Python 3.10 або вище
* pip (установник пакетів Python)

### Інструкції з налаштування
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/sunkompotus454jg/MyFences.git
   cd MyFences
   ```
2. Встановіть необхідні залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустіть застосунок:
   ```bash
   python main.py
   ```

### Збірка виконуваного файлу
Для збірки автономного виконуваного файлу та інсталятора:
1. Запустіть `build.cmd` у кореневому каталозі проєкту. Це встановить PyInstaller і збере `Nestly.exe` у папці `dist`.
2. Завантажте та встановіть Inno Setup.
3. Відкрийте `installer.iss` у Inno Setup і натисніть "Compile" (Компілювати). Це згенерує фінальний інсталятор у папці `Output`.

## Використання

*   **Створити сітку:** Натисніть правою кнопкою миші на робочому столі та виберіть "Створити сітку Nestly" або скористайтеся меню в системному треї.
*   **Додати файли:** Перетягніть файли, папки або ярлики з робочого столу безпосередньо в сітку.
*   **Зміна розміру та переміщення:** Потягніть за краї, щоб змінити розмір. Потягніть за заголовок, щоб перемістити сітку по робочому столу.
*   **Налаштування:** Отримайте доступ до меню Налаштувань із системного трею, щоб змінити мову, керувати профілями або налаштувати автозапуск.

## Архітектура

*   **Core:** Обробляє збереження даних, багатомовну підтримку, теми та керування профілями.
*   **UI:** Керує візуальним відображенням сіток, користувацькими преміальними діалоговими вікнами та анімаціями.
*   **IPC:** Керує забезпеченням єдиного екземпляра та зв'язком між екземплярами (наприклад, при створенні нової сітки з контекстного меню).
